# Order Pack Size Frontend Challenge

This repository contains the frontend code for the Order Pack Size feature of the Gymshark Challenge. The application allows users to create and manage order packs with specified sizes.

## Features

- Create order packs with customizable sizes.
- User-friendly interface for inputting pack sizes.
- Responsive design for various devices.

## Technologies Used

- React
- TypeScript
- Next.js
- Tailwind CSS

## Getting Started

To run the application locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/luk3skyw4lker/pack-order-calculator-frontend.git
   ``` 

2. Navigate to the project directory:
   ```bash
   cd pack-order-calculator-frontend
   ```

3. Run the Docker Compose command to build and start the application:
   ```bash
   docker-compose up --build
   ```

5. Open your browser and visit `http://localhost:3000` to view the application.
